Sistem Inventori
Deskripsi Project
Sistem Inventori adalah aplikasi desktop berbasis Java yang dikembangkan menggunakan Apache NetBeans IDE dan MySQL sebagai database. Aplikasi ini dirancang untuk membantu proses pengelolaan persediaan barang, data pelanggan, serta transaksi penjualan secara lebih cepat, akurat, dan terorganisir.
Melalui aplikasi ini, pengguna dapat melakukan pencatatan stok barang, mengelola data kategori dan pelanggan, melakukan transaksi penjualan, mencetak struk, serta memantau riwayat transaksi. Sistem juga mendukung pengelolaan hak akses pengguna (multi-user) sehingga administrator dan kasir memiliki kewenangan yang berbeda sesuai tugasnya.
Aplikasi ini cocok digunakan sebagai media pembelajaran, tugas akhir, kerja praktik, maupun sebagai dasar pengembangan sistem Point of Sale (POS) yang lebih kompleks.
________________________________________
✨ Fitur Aplikasi
🔐 Login Multi User
•	Login menggunakan username dan password. 
•	Hak akses berdasarkan level pengguna. 
•	Admin memiliki akses penuh terhadap seluruh menu. 
•	Kasir hanya dapat melakukan transaksi penjualan. 
________________________________________
📊 Dashboard
Dashboard menampilkan informasi penting seperti:
•	Total Data Barang 
•	Total Customer 
•	Total Kategori 
•	Total Transaksi 
•	Ringkasan aktivitas sistem 
________________________________________
📦 Manajemen Data Barang
Fitur pengelolaan data barang meliputi:
•	Menambah barang 
•	Mengubah data barang 
•	Menghapus barang 
•	Mencari barang 
•	Menampilkan stok barang 
•	Menampilkan harga barang 
________________________________________
📂 Manajemen Kategori
Memudahkan pengelompokan barang berdasarkan kategori.
Fitur:
•	Tambah kategori 
•	Edit kategori 
•	Hapus kategori 
•	Pencarian kategori 
________________________________________
👥 Manajemen Customer
Pengguna dapat mengelola data pelanggan seperti:
•	Nama Customer 
•	Nomor Telepon 
•	Alamat 
Fitur:
•	Tambah Customer 
•	Edit Customer 
•	Hapus Customer 
•	Pencarian Customer 
________________________________________
🛒 Transaksi Penjualan
Fitur transaksi meliputi:
•	Memilih customer 
•	Memilih barang 
•	Menentukan jumlah pembelian 
•	Keranjang belanja 
•	Perhitungan otomatis subtotal 
•	Perhitungan total belanja 
•	Input pembayaran 
•	Perhitungan kembalian secara otomatis 
________________________________________
📉 Pengurangan Stok Otomatis
Setiap transaksi yang berhasil disimpan akan secara otomatis:
•	Mengurangi stok barang sesuai jumlah pembelian. 
•	Menyimpan data transaksi ke database. 
________________________________________
🧾 Cetak Struk
Setelah transaksi selesai, sistem dapat:
•	Mencetak struk pembelian. 
•	Menampilkan detail transaksi. 
•	Menampilkan total pembayaran dan kembalian. 
________________________________________
📜 Riwayat Transaksi
Menyimpan seluruh transaksi yang telah dilakukan sehingga pengguna dapat:
•	Melihat histori transaksi. 
•	Mencari transaksi tertentu. 
•	Melakukan pengecekan data penjualan. 
________________________________________
🔍 Pencarian Data
Tersedia fitur pencarian pada beberapa menu seperti:
•	Barang 
•	Customer 
•	Kategori 
•	Riwayat Transaksi 
________________________________________
💾 Database MySQL
Seluruh data tersimpan secara permanen pada database MySQL sehingga:
•	Data lebih aman. 
•	Mudah dilakukan backup dan restore. 
•	Mendukung pengelolaan data dalam jumlah besar. 
________________________________________
🎯 Tujuan Project
Project ini dikembangkan untuk membantu proses pengelolaan inventori dan transaksi penjualan secara digital sehingga proses pencatatan menjadi lebih efisien, mengurangi kesalahan pencatatan manual, serta mempermudah pengelolaan data barang dan laporan transaksi.
